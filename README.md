# 🌱 Green Nest — Premium Botanical Care

> **Nurturing your space with elegance and modern care.**

Green Nest is a high-end, single-page botanical application designed for plant enthusiasts. We've transformed a standard plant shop into a premium, cinematic experience using modern web technologies and sophisticated animations.

---

## ✨ Features that "Wow"

- **🎬 Cinematic Animations:** Powered by **Framer Motion**, featuring coordinated "shared-swap" transitions for authentication and smooth, scroll-triggered reveals for the plant grid.
- **🪴 Custom Botanical Branding:** A bespoke, branded loading experience with a pulsing leaf animation that replaces generic spinners.
- **🔐 Distraction-Free Auth:** An asymmetrically designed login/signup flow with high-fidelity botanical imagery and coordinated entrance choreography.
- **🍹 Glassmorphic UI:** A state-aware, frosted-glass Navbar with personalized user profile dropdowns and smooth hover effects.
- **🌿 Expert Care Library:** Dive deep into curated plant details, care guides, and "Decor Ideas" to elevate your living space.
- **🔔 Premium Feedback:** Integrated **SweetAlert2** for professional, non-intrusive success notifications for newsletter and collection updates.
- **📱 Responsive Excellence:** Fully optimized for mobile, tablet, and desktop with a focus on fluid layout transitions.

## 🛠️ Local Cultivation

Follow these steps to grow the project on your local machine:

1. **Clone the project:**
   ```bash
   git clone https://github.com/Ali-Akash11/Green-Nest.git
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Establish Firebase Connection:**
   Create a `.env.local` file and add your Firebase credentials:
   ```env
   VITE_apiKey=Your_API_Key
   VITE_authDomain=Your_Domain
   ...
   ```

4. **Launch the environment:**
   ```bash
   npm run dev
   ```
